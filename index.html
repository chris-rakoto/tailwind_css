BUSY Bar est un dispositif multi-outil de productivité avec un écran LED pixelisé. Minuteur de concentration avec fonction de blocage des distractions sur votre télephone et PC. Entièrement personnalisable, open-source et prêt pour la maison connectée.

Outil de productivité

Bloquez les distractions, personnalisé, minuteur de concentration, synchronisation multiplateforme

Applications et intégration

Intégrés avec les logiciels tiers, intégrations avec les événements du calendrier et les appels

Support maison connectée

Connectez via le protocole Matter, compatible avec tous les appareils

Pour les développeurs

API HTTP ouverte, SDK open-source, bibliothèques Python / Go / JavaScript
